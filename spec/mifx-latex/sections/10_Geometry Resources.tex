\section{Geometry Resources}

Geometry resources represent the physical objects involved in the machining process.

These may include:

\begin{itemize}
\item part models
\item stock material
\item fixtures
\item setup assemblies
\end{itemize}

Geometry resources are stored within the \texttt{geometry} directory of the MIFX package.

Example structure:

\begin{verbatim}
geometry/
  setups/
  parts/
  stock/
  fixtures/
\end{verbatim}

The specification does not require all geometry directories to be present.

\subsection{Geometry Referencing}

Geometry resources are referenced through artifact objects contained within MIFX objects such as setup resources.

Example:

\begin{verbatim}
{
  "role": "geometry",
  "kind": "3mf",
  "path": "../../geometry/setups/set-1.3mf",
  "present": true
}
\end{verbatim}

The \texttt{path} field references the location of the geometry file within the MIFX package.

\subsection{Supported Geometry Formats}

The MIFX specification does not mandate a single geometry format.

Common formats include:

\begin{itemize}
\item STL
\item 3MF
\item STEP
\item OBJ
\end{itemize}

Implementations may support additional formats depending on their capabilities.

\subsection{Setup Assemblies}

Setup assemblies represent the full machining configuration for a setup.

Assemblies are typically stored in the \texttt{geometry/setups} directory.

\subsection{Part Geometry}

Part models represent the design geometry of the component being manufactured.

Part geometry is typically stored in the \texttt{geometry/parts} directory.

\subsection{Stock Geometry}

Stock geometry describes the initial material from which the part will be machined.

Stock models are typically stored in the \texttt{geometry/stock} directory.

\subsection{Fixture Geometry}

Fixture models represent workholding or machine fixtures used during machining.

Fixture geometry is typically stored in the \texttt{geometry/fixtures} directory.

\subsection{Tool Geometry}

Tool geometry is not stored in the global \texttt{geometry} directory.

Optional tool meshes are stored within the corresponding tool group directory as part of the Tool object.

\subsection{In-Process Geometry}

Intermediate workpiece geometry is stored within the \texttt{in\_process} directory alongside the corresponding In-Process object.
